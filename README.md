# Expense Tracker PWA

## Features
- Progressive Web App (offline support, installable)
- PHP + MySQL backend
- Android emulator compatible

## How to Run
1. Import database from `database/expense_pwa.sql`
2. Place `backend` folder inside XAMPP `htdocs`
3. Serve `frontend` using:
   ```
   npx serve frontend
   ```
4. Open in browser:
   - PC: http://localhost:3000
   - Emulator: http://10.0.2.2:3000
